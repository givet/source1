====================
软件说明
====================
该软件是需拼多多上货助手的使用。解决了拼多多上货助手批量导出后，无法批量导回批多多上货助手的问题。
由于拼多多的规则是第一张轮播图不能有文字和水印。该软件可以快速选择好无文字的轮播图后，批量导回拼多多上货助手。


====================
使用前说明
====================
1、 打开1.txt 
2、 替换toolPath=后的路径为拼多多上货助手的安装目录下downloadPic文件夹路径
3、 替换handPath=后的路径为拼多多上货助手导出图片的路径
4、 运行1.bat使用
如无法运行，请确认机子有安装java 1.6

====================
使用说明
====================
输入1 清理拼多多上货助手的安装目录下downloadPic文件夹下所有文件，慎重使用

输入2 清理图片处理目录，
这个就是拼多多上货助手导出图片的目录，每次准备开始处理前，请清空

输入3 拷贝上货助手待处理图片到处理目录(请先使用上货助手导出图片到处理目录，再运行)
在处理导出图片的时候，可以配合win7的预览视图，把符合拼多多规则的轮播图，利用重命名的方式，排到第一张。
（比如一个商户有轮播图1,轮播图2，轮播图3，如果轮播图2没有文字，那么把轮播图1直接删掉，轮播图2就排到第一张了）

输入4 全部处理完成后，一次性导回拼多多上货助手

最后，使用拼多多的上传并提交，完成商品的上货。
